# HyperTea
The Pytorch implementation of the paper HyperTea: A Hypergraph-based Temporal Enhancement and Alignment Network for Moving Infrared Small Target Detection.

# Prerequisite
python==3.10.14
pytorch==2.0.0
torchvision==0.15.1
numpy==2.1.1
With CUDA 12.4, and 1x NVIDIA 4090D.
